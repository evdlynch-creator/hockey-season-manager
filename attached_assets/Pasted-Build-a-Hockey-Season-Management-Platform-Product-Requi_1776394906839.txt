Build a Hockey Season Management Platform

Product Requirements Document (PRD) — MVP (Final)
1. Product Overview
Product Name

Inside Edge: Hockey Season Manager

Objective

Build a web application that allows hockey coaches to:

Plan and store practices
Log games and coach feedback
Track team progress across key concepts
Review past performance to improve future practices and games
Core Value Proposition

Help coaches understand if what they practice is improving game performance — and what to adjust.

5.14 Calculation Rules
Ratings stored as integers (1–5)
Converted to % using:
(score / 5) * 100
Aggregation
Simple average
Missing values excluded
Trend Logic
Up: +5% or more
Down: -5% or more
Flat: within ±5%
Weighting
Practice: 60%
Game: 40%
